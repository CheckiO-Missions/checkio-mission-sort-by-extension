Вам дана последовательность файлов в виде строк. Вам нужно отсортировать эту последовательность по расширению файла. Файлы с одинаковым расширением (или без него) должны быть отсортированы по имени.

Некоторые возможные случаи:

  • Имя файла не может быть пустой строкой;
  • Порядок сортировки: файлы без имени, файлы без расширения, файлы с именем и расширением;
  • Файлы с именами .config или config. это все имена с пустым расширением;
  • Имя файла как str1.str2.str3 имеет расширение str3 и имя str1.str2;
  • Имя файла как .str1.str2 имеет расширение str2 и имя .str1.

Ввод: {% if is_js %}Array{% else %}Список{% endif %} строк (str{% if is_js %}ing{% endif %}).

Вывод: {% if is_js %}Array{% else %}Список{% endif %} строк (str{% if is_js %}ing{% endif %}).
