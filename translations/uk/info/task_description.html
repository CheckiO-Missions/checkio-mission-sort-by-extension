Тобі даний {% if is_js %}масив{% else %}список{% endif %} файлів. Потрібно відсортувати його за розширенням файлів. Файли з однаковим розширенням (або без нього) повинні сортуватися за іменем.

Окремі можливі випадки:

  • Файл не може бути порожнім рядком;
  • Порядок сортування: файли без імені, файли без розширення, файли з обома;
  • Файли ".config" or "config." це все ім'я з порожнім розширенням;
  • Файли виду "str1.str2.str3" мають розширення "str3" і ім'я "str1.str2";
  • Файли виду ".str1.str2" мають розширення "str2" і ім'я ".str1".

Вхідні дані: {% if is_js %}Масив{% else %}Список{% endif %} рядків.

Output: {% if is_js %}Масив{% else %}Список{% endif %} рядків.
