You are given with a list of files. You need to sort this list by file extension. Files with the same extestion should be sorted by name.

Some cases about extensions:

  • Filename can't be an empty string
  • Files without extensions should go before files with extension
  • Filename ".config" has empty extenstion and name ".config"
  • Filename "config." has empty extenstion and name ".config"
  • Filename "table.imp.xls" has extesntion "xls" and name "table.imp"
  • Filename ".imp.xls" has extension "xls" and name ".imp"

Input: A list of filenames.

Output: A list of filenames.
